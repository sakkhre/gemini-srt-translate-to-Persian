
# راهنمای نصب و استفاده از Persian Subtitle Pro

این برنامه یک ابزار تخصصی برای ترجمه زیرنویس‌های فیلم به فارسی محاوره‌ای (شکسته) است که از قدرت هوش مصنوعی Gemini استفاده می‌کند.

## نیازمندی‌های سیستم
- سیستم‌عامل: ویندوز ۱۰ یا ۱۱ (۶۴ بیتی)
- پردازنده: Intel Core i3 یا بالاتر
- حافظه رم: حداقل ۴ گیگابایت
- اتصال اینترنت فعال (برای ترجمه)

## مراحل نصب (نسخه وب/React)
این برنامه در حال حاضر به عنوان یک اپلیکیشن وب پیشرفته ارائه شده است. برای اجرای محلی:
1. پوشه پروژه را باز کنید.
2. دستور `npm install` را برای نصب وابستگی‌ها اجرا کنید.
3. با دستور `npm run dev` برنامه را اجرا کنید.

## نحوه استفاده
1. **بارگذاری فایل:** روی دکمه "بارگذاری فایل SRT" کلیک کنید و فایل زیرنویس انگلیسی خود را انتخاب کنید.
2. **ترجمه خودکار:** روی دکمه آبی رنگ "ترجمه خودکار" کلیک کنید. برنامه در دسته‌های ۸ تایی سطرها را به هوش مصنوعی می‌فرستد تا از محدودیت‌ها جلوگیری شود.
3. **ویرایش دستی:** شما می‌توانید در ستون سمت راست هر سطر را به دلخواه ویرایش کنید تا طبیعی‌تر شود.
4. **ذخیره خروجی:** پس از اتمام، دکمه "ذخیره فایل نهایی" را بزنید. فایل جدید با پسوند `.fa.srt` در کنار فایل اصلی ذخیره می‌شود.

## تبدیل به فایل EXE (مخصوص توسعه‌دهندگان)
برای تبدیل این پروژه به یک فایل اجرایی ویندوز، می‌توانید از Electron استفاده کنید:
1. `npm install electron-builder --save-dev`
2. تنظیمات `package.json` را برای بیلد ویندوز انجام دهید.
3. اجرای دستور `npm run build`

## نکات مهم
- **فونت:** این برنامه از فونت "وزیرمتن" برای خوانایی بهتر فارسی استفاده می‌کند.
- **سبک ترجمه:** تمرکز اصلی روی زبان کوچه و بازاری و محاوره‌ای است تا حس تماشای فیلم در سینما به خوبی منتقل شود.
- **محدودیت سطرها:** مشکل محدودیت ۲۴ سطر در این نسخه با سیستم دسته‌بندی (Batching) بهینه شده، برطرف گردیده است.
